# Prompts Collection

This page contains a collection of prompts for GitHub Copilot.

## Documentation & Deployment

### MkDocs with GitHub Pages

**Tags:** `mkdocs`, `github-pages`, `github-actions`, `documentation`, `deployment`, `ci-cd`

**Prompt:**

```text
Make a "MkDocs" website that is built then published to "GitHub pages" for each commit with a GitHub action.
```

**Use Case:** This prompt helps you quickly set up a documentation website using MkDocs that automatically builds and deploys to GitHub Pages whenever you push changes to your repository.
